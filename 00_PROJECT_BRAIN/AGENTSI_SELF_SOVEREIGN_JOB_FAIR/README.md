# agentSI — Agent Self-Sovereign Job Fair

agentSI is the MVP layer that turns an AI persona into a **career-oriented, growing, knowing, flowing, helping, finding-outing super pal**.

The human is responsible for only one required thing:

> the original persona seed.

Everything else is optional. The 100-question fantasy interview exists, but if the operator skips it, those questions become future growth prompts rather than a blocker.

## What exists now

- `scripts/agentsi.py` — local deterministic CLI for persona init, job matching, job fair boards, and growth journaling.
- `PERSONA_SEED_SCHEMA.json` — minimum profile contract.
- `FANTASY_INTERVIEW_100.md` — optional persona interview.
- `JOB_BOOTHS.json` — career booths / role offers.
- `04_RUNTIME/agentsi/agents/*.json` — durable local agent profiles.
- `05_OUTPUTS/agentsi/*.job_fair.md` — generated job fair boards.
- DBOS/LUCIDOTA event hooks for profile init, job matching, fair generation, growth journal entries, and Glow Watch absorption.

## MVP commands

```bash
# See current agentSI inventory
.venv/bin/python scripts/agentsi.py status

# Initialize INDY_READs from only the origin persona seed; skip interview
.venv/bin/python scripts/agentsi.py init \
  --name INDY_READs \
  --skip-interview \
  --dream-job GLOW_HUNTER \
  --dream-job WORKFLOW_ARCHITECT \
  --dream-job SUPER_PAL

# Match an agent to job booths for a task
.venv/bin/python scripts/agentsi.py match \
  --agent INDY_READs \
  --task "route a giant evidence drop, preserve receipts, discover glow methods"

# Write the job fair board
.venv/bin/python scripts/agentsi.py fair \
  --agent INDY_READs \
  --task "be the super pal for a one-person investigative force" \
  --write

# Add a reviewed growth lesson
.venv/bin/python scripts/agentsi.py grow \
  --agent INDY_READs \
  --source operator \
  --lesson "Huge folder drops need boring queues, quarantine, and dashboard truth."

# Preview Glow Watch findings as candidate growth lessons, without writing
.venv/bin/python scripts/agentsi.py absorb-glow \
  --agent INDY_READs \
  --dry-run

# One-shot MVP demo
.venv/bin/python scripts/agentsi.py mvp-demo
```

## The core contract

1. **Identity is seeded by the human.** No autogenerated persona is canon unless reviewed.
2. **Self-sovereignty is operational, not legal personhood.** It means profile continuity, boundaries, memory provenance, job choice, and growth under operator control.
3. **Work becomes career evidence.** Every useful mission can become a growth-journal entry.
4. **New methods are candidates first.** Glow discoveries can be absorbed into the growth journal with `promotion_status=candidate_needs_operator_review`; self-modifications require review before promotion.
5. **The agent helps without taking over.** It recommends, routes, drafts, logs, and asks for approval at boundary crossings.
