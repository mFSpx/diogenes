# DIOGENES Glossary

## DIOGENES

Sovereign, encrypted, ontology-first investigative cognition system.

## Indy_Reads

The current user-facing assistant/persona for `northern.strike`.

## Clawd / claudecode

Rust CLI/interface shell intended to host the user interaction layer.

## CKDOG1 / doggystyle

Kernel/backend target. Holds ontology, state, routing, proofs, and symbolic compute.

## 414 Words of Power

Creator-canonical immutable base primitives. Public release must not make them user-editable.

## 469 Base Ontology

Wider base ontology that future architecture must work with.

## Village / Villagers

5,000 active soul UUIDs representing the current most important symbols/focus items.

## Symbol

Any ontology-addressable object: word, phrase, emoji, document, sentence, phone number, entity, workflow, model, source, or artifact.

## Ternary State

Contextual triadic compute state. Not restricted to `+1 / 0 / -1`.

Canonical modal labels currently include:

- `+1 SANTA`
- `0 BEEP BOOP`
- `-1 KRAMPUS`

These are important internal state labels. Their meaning is context-dependent and will be specified further by northern.strike.

## Krampus Express

Canonical internal investigation/analysis loop name. Details pending user specification; do not demote it to a throwaway metaphor.

## Records Office

The system-of-record layer: encrypted CAS/vault, hashes, provenance, citations, decisions, and traceability.

## Fastlane / Slowlane

Fastlane keeps the user interaction responsive. Slowlane handles heavy ingest, embedding, graph updates, learning, and workflow processing.

## Subtle Knife Protocol

Graduated 1-5 autonomy protocol. Exact contents pending reingestion/specification.
