# Source Repositories

This directory contains LUCIDOTA source trees and separately maintained kernel source.

- `claudecode`: LUCIDOTA-owned interface fork. This is part of LUCIDOTA now.
- `doggystyle`: CKDOG1 kernel-only repo. Kernel changes belong there and stay separately maintained.

The LUCIDOTA umbrella repo records project plans, orchestration, verification, integration notes, and the custom interface fork.
